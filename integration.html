Integration

Проект, грамотно поделенный на две части - отдельно frontend(javascript) и отдельно backend(java17, spring mvc) Взаимодействие, как и положено, осуществлено посредством диалога API. Backend взаимодействует с БД, откуда средствами hibernate забирает данные и через rest-контроллеры имеет возможность "отправлять" эти данные тому, кто их запросит.

Frontend с помощью javascript фетчит нужный URL, получая в респонсе json с данными. После обработки, информация динамически выводится на HTML страницу. Оформил в минималистическом стиле. Добавил поиск, динамически добавил слайдеры с фотографиями для каждого магазина.

Слева код spring rest контроллера, который "отдаёт" данные, а справа - javascript файл, "читающий" данные, и динамически выводящий их на экран

В браузере это выглядит так:

Здесь представлен список всех цветочных магазинов. Оценка каждого магазина, приходящая с бэкэнда преобразуется в звёздочки для наглядности. Так же каждому магазину программно назначены 3 фотографии, которые располагаются в слайдерах. "Листать фото" - перелистывание слайдера. Фотографии хранятся на фронтэнде, а в базе данных на бэкэнде располагаются лишь названия фотографий. Все названия разные, но построены по одной схеме:

Так же, реализована строка поиска - магазин можно найти, зная 3 символа из его названия. Поиск отзывчивый, осуществляется в момент ввода каждого нового символа - поэтому нет надобности совершать доп. действия вроде нажатия подтверждения, или кнопки ENTER.

Поиск основан на методе indexOf(), который возвращает -1, если в строке нет искомой подстроки. Всем элементам, кроме найденного прибавляется класс 'hide' - который скрывает контент с экрана (display:'none')
